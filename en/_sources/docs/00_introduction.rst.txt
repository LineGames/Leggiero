============

Leggiero
========

Leggiero is a game engine, or framework, mainly targeted iOS and Android platform.
It adopts C++ as a common development language to support both platforms, and has been designed for a lightweight and thin engine with direct access for OpenGL ES APIs.

Leggiero was developed for a game named `Protocol:hyperspace Diver <https://www.facebook.com/P.hyperspaceDiver/>`_ from NextFloor Basement Lab. (currently, Line Games).
After some further developing for improvement, it became an open source software.
