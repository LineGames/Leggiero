Introduction¶

Leggiero¶

Leggiero는 iOS와 Android 플랫폼을 주 대상으로 하는 게임 엔진, 혹은 프레임워크입니니다. 다양한 플랫폼 지원을 위해 공용 개발 언어로 C++를 사용하며, 가벼운 엔진을 지향하여 OpenGL ES API를 직접 사용하여 게임 그래픽 프로그래밍을 할 수 있도록 도와줍니다.

Leggiero는 2017년 발표된 넥스트플로어 지하연구소(현: 라인게임즈)의 게임 Protocol:hyperspace Diver 의 개발 과정에서 만들어졌으며, 이후 구조 개선 작업을 거쳐 오픈 소스로 공개되었습니다.
